Bab 13 Regresi Linear Berganda

13.1 Konsep Dasar

Regresi linear berganda melibatkan lebih dari satu variabel independen untuk memprediksi variabel dependen. \[ Y = a + b_1X_1 + b_2X_2 + \dots + \epsilon \]

13.2 Studi Kasus dengan R

Memprediksi Harga Rumah berdasarkan Luas Tanah dan Jumlah Kamar.

harga <- c(500, 700, 600, 800, 900)
luas <- c(100, 150, 120, 160, 200)
kamar <- c(2, 3, 2, 4, 5)

# Model Regresi Berganda
model_berganda <- lm(harga ~ luas + kamar)
summary(model_berganda)
