Bab 14 Analisis Statistik Multivariat Interdependensi

14.1 Konsep Dasar

Analisis multivariat interdependensi tidak membedakan variabel menjadi dependen dan independen. Tujuannya adalah untuk mencari struktur dasar dari sekumpulan variabel. Contoh: Analisis Faktor dan Principal Component Analysis (PCA). Biasanya digunakan untuk mereduksi dimensi data (menyederhanakan banyak variabel menjadi beberapa faktor utama).

14.2 Studi Kasus dengan R

Mereduksi 5 indikator kepuasan menjadi faktor-faktor utama.

# Simulasi Data
set.seed(123)
data_survei <- matrix(rnorm(100*5), ncol=5)
colnames(data_survei) <- c("Q1", "Q2", "Q3", "Q4", "Q5")

# Principal Component Analysis
pca_result <- prcomp(data_survei, scale. = TRUE)
summary(pca_result)
## Importance of components:
