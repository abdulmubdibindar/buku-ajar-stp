15 Regresi Linear Sederhana

15.1 Konsep Dasar

Regresi linear sederhana digunakan untuk memprediksi nilai satu variabel dependen (\(Y\)) berdasarkan satu variabel independen (\(X\)). Persamaannya: \[ Y = a + bX + \epsilon \]

15.2 Studi Kasus dengan R

Memprediksi Pengeluaran berdasarkan Pendapatan.

pendapatan <- c(5, 6, 7, 8, 9, 10, 11)
pengeluaran <- c(3, 4, 3.5, 5, 6, 6.5, 7)

# Membuat Model Regresi
model <- lm(pengeluaran ~ pendapatan)
summary(model)
##

##

Interpretasi: Nilai Intercept (\(a\)) dan Slope (\(b\)) menentukan persamaan garis prediksi. \(R^2\) menunjukkan seberapa baik model menjelaskan variasi data.
